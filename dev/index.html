Apple loves Isotope

Исправлен баг со схлопнутым изотопом на маке.

Изотоп инициализируется по обертке с классом .isotope-wrap. Внутри обертки в обязательном порядке должен присутствовать один элемент с классом .isotope по которому работает сам скрипт. Каждый элемент изотопа определяется классом .isotope-item который может иметь атрибут с категорией data-filter='CATEGORY_NAME', по которой в дальнейшем осуществляется фильтрация. Фильтры задаются посредством любых элементов внутри обертки с атрибутом data-isotope-filter='FILTER_NAME'. Также элемент изотопа .isotope может иметь атрибут с леяутом data-isotope-layout. Доступные леяуты и остальной функционал находятся в документации изотопа.
